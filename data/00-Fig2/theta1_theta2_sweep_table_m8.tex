\section*{Supplementary Table S2. Parameter sweep for eight magnets per layer}
\noindent
This table summarizes the theta-angle sweep for arrays with eight magnets in each layer
($m=8$). For each combination of $\theta_1$, $\theta_2$, layer number, and magnet
radius, the table reports the optimized maximum field along the three Cartesian
axis pairs, $B_x^{\max}$, $B_y^{\max}$, and $B_z^{\max}$, together with the
minimum value across the six target directions. All magnetic-field values are
reported in millitesla (mT). Entries marked with ``--'' correspond to invalid
geometries for which no optimized magnetic-field value was available.

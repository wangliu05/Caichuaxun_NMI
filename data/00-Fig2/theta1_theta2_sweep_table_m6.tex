\section*{Supplementary Table S1. Parameter sweep for six magnets per layer}
\noindent
This table summarizes the theta-angle sweep for arrays with six magnets in each layer
($m=6$). For each combination of $\theta_1$, $\theta_2$, layer number, and magnet
radius, the table reports the optimized maximum field along the three Cartesian
axis pairs, $B_x^{\max}$, $B_y^{\max}$, and $B_z^{\max}$, together with the
minimum value across the six target directions. All magnetic-field values are
reported in millitesla (mT).
